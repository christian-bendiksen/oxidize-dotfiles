# oxidize-dotfiles
Dotfiles for AerynOS with a atomic theme switcher powered by [oxidize](https://github.com/christian-bendiksen/oxidize-theme).
Currently work in progress!

## Install
```bash
curl -fsSL https://raw.githubusercontent.com/christian-bendiksen/oxidize-dotfiles/main/install.sh | bash

What's included
- Compositors — MangoWM
- Bar — Waybar
- Terminal — Alacritty
- Launcher — Walker
- Editor — Helix
- Monitor — btop
- Many themes, switchable at runtime

Usage

oxidize set <theme>   # switch theme
oxidize wallpaper     # cycle wallpaper
