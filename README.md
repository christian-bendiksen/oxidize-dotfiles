# oxidize-dotfiles

Wayland desktop dotfiles for AerynOS with a atomic theme switcher powered by [oxidize](https://github.com/christian-bendiksen/oxidize-theme).

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/christian-bendiksen/oxidize-dotfiles/main/install.sh | bash
```

## What's included

- **Compositors** — MangoWC
- **Bar** — Waybar
- **Terminals** — Alacritty
- **Launcher** — Walker
- **Editor** — Helix
- **Monitor** — btop
- Many themes, switchable at runtime

## Usage

```bash
oxidize set <theme>   # switch theme
oxidize wallpaper     # cycle wallpaper
```
